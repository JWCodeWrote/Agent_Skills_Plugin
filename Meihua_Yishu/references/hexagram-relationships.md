# 卦象关系深度指南：应、承乘、互卦、錯卦、綜卦、变卦
*基於周易384爻統計分析*
# 快速参考：六种关系的作用
 关系  定义  作用  何时看
 应  初↔四、二↔五、三↔上  有沒有外援  问合作、求助
 承乘  相鄰兩爻上下关系  上下关系顺不顺  问職場、輩分
 互卦  爻2-5组成的卦  內在本质（最重要！） 每次必看
 錯卦  阴阳全反  互補視角  看对立统一
 綜卦  上下顛倒  对方視角  问感情、談判
 变卦  动爻变化後的卦  发展趨勢  有动爻时
# 一、应（对应爻）— 看外援
# 对应关系
初爻 ←→ 四爻
二爻 ←→ 五爻  ← 最重要（君臣相应）
三爻 ←→ 上爻
# 判断规则
- 一阴一阳 = 有应（有人幫）
- 兩阳或兩阴 = 无应（孤軍奮戰）
# 統計发现（重要！）
 应位  有应吉率  无应吉率  效果
 二↔五  68.8%  51.6%  +17.2%
 初↔四  46.9%  43.8%  +3.1%
 三↔上  34.4%  28.1%  +6.3%
# 占卜应用
- 问事业：看体卦与用卦的二五爻是否有应
- 二五有应：有贵人、資源到位、合作顺利
- 二五无应：需自力更生、缺乏外援
# 二、承乘 — 看上下关系
# 定义
承 = 下爻承托上爻
乘 = 上爻騎乘下爻
# 四种组合
 类型  吉率  凶率  意义
 阳乘阴  43.8%  25.0%  剛統柔，最顺
 阴承阳  35.0%  35.0%  柔顺剛，平衡
 阳承阴  中等  中等  剛在柔下
 阴乘阳  22.5%  50.0%  柔凌剛，最危
# 化解「阴乘阳」
三者具二即可转吉：得中 + 得尊位 + 合时义
# 占卜应用
- 看体卦动爻与相鄰爻的阴阳关系
- 阴乘阳出现时：提醒问卦者注意上下关系、權力结结构
# 三、互卦 — 看內在本质（最重要！）
# 计算方法
下互卦 = 爻 2, 3, 4
上互卦 = 爻 3, 4, 5
# 統計发现
- 影响力系数：0.7315（其他关系只有 0.19 基線）
- 互卦是所有关系中影响力最强的！
# 时间意义
- 用卦 = 近期（当前狀態）
- 互卦 = 中期（发展过程、真正走向）
- 变卦 = 遠期（最終结果）
# 占卜应用
- 每次解卦必看互卦
- 本卦吉但互卦凶 = 表面好但有隱患
- 本卦凶但互卦吉 = 困难中有出路
- 互卦生体 = 过程中有助力
- 互卦克体 = 过程中有阻礙
# 四、錯卦 — 看互補面
# 定义
所有爻阴阳互換
# 統計发现
- 吉凶相反率：只有 28.1%
- 不是「对立」而是「互補」
# 常见錯卦对
 本卦  錯卦  关系
 乾  坤  創造 ↔ 承載
 泰  否  通泰 ↔ 閉塞
 既濟  未濟  完成 ↔ 未完
# 占卜应用
- 理解事物的另一面
- 看到现象的互補性质
# 五、綜卦 — 看对方視角
# 定义
整卦 180 度翻转
# 爻位对应
本卦初爻 ↔ 綜卦上爻
本卦二爻 ↔ 綜卦五爻
本卦三爻 ↔ 綜卦四爻
# 統計发现
- 吉凶相近率：64.3%（最高相似度）
- 8卦自对稱：乾、坤、頤、大过、坎、离、中孚、小过
# 占卜应用
- 问感情/談判：把卦倒过来看对方視角
- 你在初爻 → 对方在上爻
- 理解对方处境後更容易找到共识
# 六、变卦 — 看发展趨勢
# 计算方法
动爻阴阳互換後形成新卦
# 384变卦統計发现
 指标  数值
 之卦趨勢向上 (↑)  38.5%
 之卦趨勢向下 (↓)  38.5%
 之卦趨勢持平 (→)  22.9%
# 阴爻 vs 阳爻变化（重要！）
 爻性  变好率  变差率  占卜建議
 阴爻动  41.1%  35.9%  变化較安全
 阳爻动  35.9%  41.1%  需謹慎评估
规律：阴爻变动後更容易变好；阳爻变动後更容易变差（阳极必反）
# 各爻位变化特性（重要！）
 爻位  凶→吉  吉→凶  占卜建議
 三爻  8例（最多）  6例  困境中有转机，不必过慮
 初爻  6例  15例  起步者变动风險高
 二爻  3例  14例  得中者不宜妄动
 五爻  2例  11例  尊位者不宜妄动
规律：
- 三爻本「多凶」，但变化後最易转好（窮则变，变则通）
- 二五爻本「多吉」，但变动後容易变差（守成者慎动）
# 臨卦是关鍵转折点（重要！）
 情況  幅度  意义
 变成臨卦  +0.83  好兆头，接近成功
 离开臨卦  -0.83  需警惕，离开好位置
臨卦特性：二阳在下、四阴在上，代表「贵人降臨」「接近成功」

 本卦  动爻  之卦  趨勢
 归妹  四爻  臨  +0.83

吉爻变凶（好景不长）：
 本卦  动爻  之卦  趨勢
 臨  五爻  節  -0.83
 臨  四爻  归妹  -0.83
 臨  初爻  师  -0.67
 比  二爻  坎  -0.67
 鼎  上爻  恆  -0.67
# 占卜应用
- 阴爻动 → 較安全
- 阳爻动 → 需謹慎
- 三爻动且本凶 → 不必过慮，变後可能转好
- 二五爻动且本吉 → 考慮是否真需要变动
- 变成臨卦 → 好兆头
- 离开臨卦 → 需警惕
- 动爻凶但变卦好 = 窮则变，变则通
- 动爻吉但变卦差 = 居安思危

# 范例

通关化解
体用相克时，互卦或变卦有通关元素可緩解凶象。
通关=能同时与兩個相克元素产生生关系的第三元素
通关对照表
克关系|通关|原理|卦象
金克木|水|金生水水生木|乾兌→坎→震巽
木克土|火|木生火火生土|震巽→离→坤艮
土克水|金|土生金金生水|坤艮→乾兌→坎
水克火|木|水生木木生火|坎→震巽→离
火克金|土|火生土土生金|离→坤艮→乾兌
通关应用
1确定体用关系（克关系则繼續）
2查看互卦变卦是否有通关元素
3有通关凶象減輕，无则維持
通关歌诀
金木相爭水通关，木土相克火当先。土水交戰金为媒，水火不容木来援。火金相搏土調停，五行通关化凶煞。
